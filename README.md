# Генератор задач №6 ЕГЭ (база)

Веб-приложение генерирует задачи по теме «Доли и проценты» в формате ЕГЭ (база, №6). Генерация полностью алгоритмическая, работает офлайн и не использует LLM/нейросети в рантайме.

## Запуск

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Использование seed

* Поле **Seed** позволяет воспроизводить задачу. Если указать одно и то же число, будет получена идентичная задача и решение.
* Пустое поле означает случайную генерацию.

## Типы задач

1. Отношение a:b → процент одной части.
2. Часть от целого (x% от N).
3. Процентное изменение (было/стало → найти %).
4. Увеличилось/уменьшилось в k раз → найти процент изменения.
5. Скидка/наценка в процентах.
6. Две последовательные скидки/наценки.
7. Удержание/налог (до/после удержания).
8. Новая цена = m от старой → найти процент изменения.

## Тесты

```bash
npm run test
```

Тесты проверяют, что для фиксированного seed генерация стабильна и ответ совпадает с вычислением по параметрам.
